# Select element

## Example

<fieldset>
    <label for="fruits">Fruits</label>
    <select name="fruit" id="fruits">
        <option value="apple">Apple</option>
        <option value="strawberry" selected>Strawberry</option>
        <option value="grapes">Grapes</option>
    </select>
</fieldset>
